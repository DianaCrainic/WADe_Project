1 Introduction

This user guide is designed to help you get started with the Cryptocurrency Knowledge Management System, a modular web system for managing and expressing knowledge about cryptocurrencies and their metadata. The system queries information via a SPARQL endpoint and provides it in HTML+RDFa and JSON-LD formats, as well as multiple visualizations.

  3. Using the QR code, by scanning with you mobile device:

2.2 Data Formats

Exposed information are available in the following formats:

  1. HTML+RDFa
  2. JSON-LD

2.3 Visualizations

The system provides multiple visualizations of the data to help you understand the relationships between cryptocurrencies and their metadata. To view the visualization page, click on the Visualizations button at the top of the page, on which 2 charts will be shown for all the cryptocurrencies. The first one is a pie-chart, representing the Number of cryptocurrencies by protection scheme, and the second one is a time series chart for the Number of cryptocurrencies by founded date. Also, if you want to see the bar-charts with the information about the cryptocurrencies of each page, scroll down, and you will see one for the Total coins property and one for the Block time property.

3 User roles

There are two types of user in this application: the unauthenticated user and the administrator.

The unauthenticated user can:

  • view the cryptocurrencies paginated list
  • access the information rearding a cryptocurrency
  • export the cryptocurrencyin JSON-LD format
  • view the news regarding the cryptocurrency
  • access the page with the visualizations

The administrator can do the following actions, in addition to those that the unauthenticated user can do:

  • create a cryptocurrency
  • update a cryptocurrency
  • delete a cryptocurrency
  • create a piece of news
  • update a piece of news
  • delete a piece of news

Now, you can create your own cryptocurrency! Press the Create Cryptocurrency button in order to create one.

Please fill in the fields in the form.

  1. symbol represents the name of the cryptocurrency
  2. description is about some words about the cryptocurrency
  3. website represents the website of the cryptocurrency
  4. source represents the source of the cryptocurrency
  5. block reward represents the block reward of the cryptocurrency
  6. total coins represents the total coins of the cryptocurrency
Next, once the fields are filled, press the Save button in order to finally create the cryptocurrency.

In conclusion, the Cryptocurrency Knowledge Management System is a versatile and user-friendly platform for managing and expressing knowledge about cryptocurrencies and their metadata. The multiple visualizations provided within the system offer a unique and insightful way to understand and analyze the relationships between cryptocurrencies and their metadata.
